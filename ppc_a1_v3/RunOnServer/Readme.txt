1. Modify the Java source code to complete the coding assignment.
2. Export a Runnable Jar to this folder with "benchmark.jar" as the name.
3. Send this folder to machine-learning.di.fc.ul.pt
4. Submit a job to execute your program (sbatch run.sh)
5. Wait for the job to complete.
6. Copy records.csv to this folder on your machine
7. Run pip install -r requirements.txt
7. Run python3 analysis.py to obtain the plot.
